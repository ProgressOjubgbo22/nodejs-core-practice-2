# Node.js HTTP & Directory Practice

This repository contains hands-on practice with Node.js core concepts, focusing on directories and HTTP handling.

## What I am learning
- **Working with directories**: creating, reading, renaming, deleting, and checking existence
- **HTTP handling**:
  - Handling **GET** requests
  - Handling **POST** requests
  - Working with **query parameters**
- Using the Node.js **fs** and **http** modules to build foundational understanding

## Purpose
- To reinforce Node.js fundamentals through practical examples
- To build confidence handling files, folders, and HTTP requests
- To create a reference for my ongoing learning journey
